Jak zacząć?

  1. Musimy mieć zainstalowaną Javę 1.6 (najlepiej JDK)
  2. Ściągamy ze strony Clojure (http://clojure.org/) plik clojure-1.x.0.jar
  3. Uruchamiamy przez java -jar clojure-1.x.0.jar
  4. Pojawia się prompt REPL'a (Read Eval Print Loop)
  5. Bardziej zaawansowany REPL uzyskujemy ściągając Leiningen (http://leiningen.org/)
